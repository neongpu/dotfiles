# neongpu/dotfiles

These are the dotfiles I use in my daily **Arch/Hyprland** setup. To use them, clone this repo and then copy the folders into your ```~/.config``` directory.

Alternatively, create a backup of your ```~/.config``` folder, remove the original, and then create a syslink between the this repo's config and your config folder.

```
sudo ls -s ~/repos/dotfiles/.config ~/.config
```

A [.gitignore](.config/.gitignore) file is included to make sure only specific rice files get included in each commit.
